# Note Ubuntu

* Plugins
Open vim and run
```php
:PluginInstall
```

* Ctags
Install ctags
```php
sudo apt-get install ctags
```

* Vim-nox
The vim-nox for neocomplete support
```php
sudo apt-get install vim-nox
```

* Powerline
For nice appereance
Follow powerline installation at https://powerline.readthedocs.org/en/master/installation/linux.html
